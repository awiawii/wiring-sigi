This CD-ROM contains a technical documentation for a plant or a machine.

System Requirements
-------------------------------------
Operating system: Microsoft Windows 98/, NT 4.x, 2000, XP
Browser: Microsoft Internet Explorer or Netscape Navigator version 4.0 or higher with activated Java, and Acrobat Reader version 5.0 or higher.
We have tested the documentation using Microsoft Internet Explorer 5 and Acrobat 5.
You can download Adobe Acrobat(r) Reader(r) free at
http://www.adobe.de/products/acrobat/readermain.html

The documentation runs fully from the CD-ROM. No installation required. 
To increase performance, we recommend that you copy all data to a hard disk, preserving the original folder/path structure.
If you use the WIN XP, sp2 operating system, running the documentation from a hard disk may result in unwanted messages.
Such messages may be ignored.

Processor: 233 MHz or better
Memory: minimum 128 Mbyte RAM (256 Mbyte recommended)
Graphic card: minimum 65535 colours
Screen resolution: minimum 800*600
Hard drive: no space required
CD-ROM drive, minimum 20x speed
Mouse or similar pointer device


Installation - Uninstallation
----------------------------------------
No installation and thus no uninstallation required!
You only need to have a web browser and the Acrobat Reader installed.
Make sure the "Display PDF in browser" box is checked in your Acrobat Reader settings (Edit --> Preferences --> Internet).


Start
---------
If the CD-ROM does not start automatically after it has been inserted into the drive, you can start the START.BAT file directly or open the  ....\index.htm file with a web browser.
