Die Ihnen vorliegende CD-Rom enthält eine technische Dokumentation zu einer Anlage oder Maschine.

Systemvoraussetzungen
-------------------------------------
Betriebssystem: Microsoft Windows 98/, NT 4.x, 2000, XP
Browser: Microsoft Internet Explorer oder Netscape Navigator ab 4.0 mit aktiviertem Java und Acrobat Reader 5.0 oder höher.
Getestet wurde die Dokumentation von uns unter dem Microsoft Internet Explorer 5 und Acrobat 5.
Sie finden den Adobe Acrobat® Reader® zum kostenlosen download unter
http://www.adobe.de/products/acrobat/readermain.html

Die Dokumentation läuft vollständig von CD. Es ist keine Installation notwendig. 
Aus Performance - Gründen wird eine Kopie aller Dateien in der originalen Pfadstruktur auf Festplatte empfohlen.
Hierbei kann es jedoch unter dem betriebssystem WIN XP, sp2, zu unerwünschten Meldungen kommen.
Die Meldungen können Sie ignorieren.

Prozessor: 233 MHz oder besser
Hauptspeicher: mindestens 128 Mbyte RAM (empfohlen 256 Mbyte)
Grafikkarte: mindestens 65535 Farben
Bildschirmauflösung: mindestens 800*600
Festplattenspeicher: kein Platz notwendig.
CD-Rom-Laufwerk mit mindestens zwanzigfacher Geschwindigkeit
Maus oder vergleichbares Zeigewerkzeug


Installation - Deinstallation
----------------------------------------
Es ist keine Installation und damit keine Deinstallation notwendig!
Lediglich ein Web - Browser und Acrobat-Reader müssen installiert sein.
Achten Sie bei der Acrobat - Reader Einstellung auf den Eintrag 
/ OPTIONEN / WEB-BROWSER-OPTIONEN / 


Start
---------
Sollte der Auto - Start nach dem Einlegen der CD in das Laufwerk nicht gelingen, können Sie
die Datei  START.BAT direkt starten oder mit einem Browser die Datei  ....\index.htm öffnen.
